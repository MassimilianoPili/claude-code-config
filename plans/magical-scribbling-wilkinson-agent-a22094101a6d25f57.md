# Free Pixel Art Assets for SNES-Style Top-Down RPG (Phaser.js)

Research results for the best free pixel art tilesets, character sprites, and UI assets
for a 2D top-down RPG portfolio game in SNES/16-bit style.

---

## RECOMMENDED STRATEGY

For a cohesive SNES/Chrono Trigger aesthetic, the best approach combines:

1. **Primary tileset**: Pipoya 32x32 (outdoor + indoor) OR Mana Seed 16x16 (upscale to 32x32)
2. **Character sprites**: Pipoya 32x32 characters (64 characters, 4-dir walk) + LPC Generator for custom NPCs
3. **UI/Dialogue**: Mixel v1.5 UI pack (matching style) + Tiny Speech Indicators
4. **Effects/Extras**: OpenGameArt CC0 portals + Elthen portal sprites

### Phaser.js Bitmap Font Implementation
```javascript
// Load bitmap font (BMFont format: .png + .xml)
this.load.bitmapFont('rpg-font', 'font.png', 'font.xml');

// Use in scene
this.add.bitmapText(100, 100, 'rpg-font', 'Hello World', 16);

// Retro font from spritesheet
const config = {
    image: 'font-sheet',
    width: 8,
    height: 8,
    chars: Phaser.GameObjects.RetroFont.TEXT_SET1,
    charsPerRow: 16
};
this.cache.bitmapFont.add('retro', Phaser.GameObjects.RetroFont.Parse(this, config));
```

**Font Generation Tools**:
- **SnowB BMF** (online): https://snowb.org/en/
- **BMFont** (Windows): http://www.angelcode.com/products/bmfont/

## MY TOP RECOMMENDATION

**For a Phaser.js web-based RPG portfolio game targeting SNES/Chrono Trigger aesthetic:**

**Go with Option B (Pipoya ecosystem)** as the primary base, supplemented with specific assets:

1. **Pipoya tileset + characters** -- 32x32 native, huge variety, clean style, easy licensing
2. **Mixel UI pack** -- Matching style panels, buttons, health bars, dialogue frames
3. **Tiny Speech Indicators** -- CC0 exclamation/question/speech bubbles for NPC interaction
4. **Elthen Portal Sprites** -- For teleportation/transition effects
5. **m5x7 pixel font** -- Clean, readable, works great in Phaser bitmap text
6. **LPC Generator** for custom hero character (if Pipoya's 64 characters are not enough)

This combination gives you outdoor tiles, indoor tiles, 64+ characters, UI, indicators, portals,
and fonts -- everything needed for a complete RPG portfolio game.

If SNES authenticity is the absolute top priority and you do not plan to use any AI-generated
content in your game, consider **Mana Seed (Option A)** instead -- it is unmatched for that
specific aesthetic, though it requires upscaling from 16x16 to 32x32.
